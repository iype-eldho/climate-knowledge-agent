# Project Scope

Goal:
Build an LLM-powered research assistant that answers scientific questions
about drought, rainfall variability, and climate extremes using research papers.

Initial capability:
- ingest PDFs
- retrieve relevant text
- answer grounded questions

Example questions:
- What causes flash drought?
- How is satellite rainfall bias corrected?
- What indices are used for drought monitoring?

Phase 1 goal:
Ugly but working RAG pipeline via CLI.
